# Démo : Linter (ESLint) et formater (Prettier)

## Lancer le projet

~~~bash
npm install
~~~

## Appliquer le linter

~~~bash
npx eslint index.js
~~~

## Appliquer le formater

~~~bash
npx prettier . --write
~~~

## Références

- [ESLint](https://eslint.org/)
- [Prettier](https://prettier.io/)
